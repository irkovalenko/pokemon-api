# Description

This is a projected created with usage of <a href="https://pokeapi.co/" target="_blank">PokeApi</a>. The pokemon data is obtained via api call and seeded to the database. Pokemons from API cannot be edited but the user can also add/edit/delete his own pokemons (full crud functionality is supported). A user can comment the pokemons. An admin can ban pokemons so they are not visible in dashboard.
